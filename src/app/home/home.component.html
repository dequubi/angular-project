Требования

Редактор и просмотр элементов на отдельных страницах.

Элемент состоит из наименования, описания, даты создания и даты выполнения.

На странице редактора должно быть реализовано следующее:

  1. Добавление элемента:
    1. Добавление необходимо делать с помощью popup состоящего из полей:
      1. Название.
      2. Дата выполнения (Дата + Время).
      3. Описание.
    2. При создании элемента автоматически должно заполняться поле – Дата создания (Дата + Время).
  2. Копирование элемента.
  3. Удаление элемента.

На странице просмотра:

  1. Вывод списка созданных элементов с датой их создания.
  2. В заголовке списка должны быть названия колонок: наименование, дата и время создания, дата и время выполнения.
  3. Выпадающее меню у каждого элемента, предоставляющее возможность переместить элемент выше или ниже по списку.
  4. При нажатии на элемент открывается popup по аналогии с пунктом 1.1 блока выше только на просмотр полей.

Бонусы:

  1. При достижении даты выполнения будет появляться toaster-уведомление с информацией об элементе.
  2. Валидация и отображение ошибок на полях формы создания.
  3. Локализация интерфейса (русский, английский).
  4. В разделе просмотра реализовать фильтрацию по полям: Наименование, Дата выполнения с, Дата выполнения по.
